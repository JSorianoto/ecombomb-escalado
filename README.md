# 💣 EcomBomb · Guía de Escalado Intraday

Guía interactiva para alumnos de la academia **EcomBomb**: aprende a tomar decisiones de escalado de presupuesto en Meta Ads, orientado a negocios COD (contrareembolso).

## 🌐 Ver online

👉 **[https://jsorianoto.github.io/ecombomb-escalado/](https://jsorianoto.github.io/ecombomb-escalado/)**

## 📚 Contenido

5 módulos progresivos:

1. **Principios** — Los 5 fundamentos del escalado intraday
2. **Árbol de decisión interactivo** — Clica y descubre qué hacer en cada escenario
3. **CBO vs ABO** — Cuándo usar cada una
4. **Casos reales** — 3 escenarios con números reales
5. **Errores comunes + Quiz** — Evita los errores más costosos y autoevalúate

## 🎓 Para alumnos

- Pensado para **negocios COD ya vendiendo** (no testing)
- Lenguaje claro, sin tecnicismos innecesarios
- Aplicable inmediatamente en tus campañas

## 📱 Cómo usar

1. Abre el link
2. Recorre los módulos en orden la primera vez
3. Vuelve al árbol de decisión cada vez que dudes qué hacer
4. Haz el quiz final para comprobar que lo has pillado

## 🛠️ Tecnología

Una sola página HTML. Sin build step. Sin dependencias. Funciona offline tras primera carga.

- Tailwind CSS vía CDN
- Vanilla JavaScript
- Mobile-first
